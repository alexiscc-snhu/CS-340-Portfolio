# CS-340 Portfolio

## How do you write programs that are maintainable, readable, and adaptable?

I write maintainable programs by organizing code into separate modules, using meaningful variable and function names, and adding comments where appropriate. The CRUD Python module developed in Project One allowed me to separate the database operations from the dashboard. This made the code easier to reuse in Project Two without rewriting database logic. In the future, this module could be reused in other applications that need to access the same MongoDB database.

## How do you approach a problem as a computer scientist?

I approach problems by first understanding the project requirements and then breaking the work into smaller tasks. For this project, I first developed the CRUD module, then built the dashboard, and finally added the interactive filters, chart, and map. This project required integrating multiple technologies, which differed from previous assignments that focused on individual programming concepts. In the future, I would continue using this step-by-step approach to design databases and applications that meet client requirements.

## What do computer scientists do, and why does it matter?

Computer scientists develop software that solves real-world problems by organizing, processing, and presenting data in meaningful ways. This dashboard helps Grazioso Salvare quickly locate animals that meet specific rescue criteria instead of manually searching thousands of records. By providing interactive filters, charts, and maps, the dashboard improves efficiency and supports better decision-making.
